Peter Attia, last updated March 22, 2020¶

This notebook is used to explore the time dependence of SEI growth. This notebook is identical to "SEI time dependence.ipynb", save for one difference: it uses profile likelihood confidence intervals instead of asympototic approximate CIs. However, because the differences were negligible, I present the standard approach in the paper for simplicity. This notebook can be run to compare the outputs from the two methods.

Preliminaries¶

Set $\alpha$ for 95% CI:

Create helper functions for the three functional forms under investigation:

  • $y = ax^{0.5}+b$; ("t_onehalf" function)
  • $y = ax^{b}$; ("power1" function)
  • $y = ax^{b}+c$; ("power2" function)
In [4]:
def t_onehalf(x, a, b):
    return a * x**0.5 + b

def power1(x, a, b):
    return a * x**b

def power2(x, a, b, c):
    return a * x**b + c

Create tval wrapper:

In [5]:
def tval(alpha, n, p):
    dof = max(0, n - p) # degrees of freedom
    tval = t.ppf(1 - alpha/2, dof)
    return tval

Create functions to calculate CIs using the F test. These asymmetric profile confidence intervals are more representative than the asymptotic approximate symmetrical 95% CIs. In practice, however, I found that the two CIs produce very similar results. See these references for the basic procedure:

  • Motulsky and Ransnas (justification for F test)
  • Kemmer and Keller: good illustrations of the pitfalls of linearly approximated 95% CIs
  • Wikipedia
  • GraphPad profile likelihood confidence intervals
  • GraphPad F test

These results were cross-checked with R's anova function.

In [6]:
def calculate_power1_CIs_from_F_test(xdata, ydata, maxfev=500000, plot=False):
    
    # Get baseline fit
    popt, pcov = curve_fit(power1, xdata, ydata, maxfev=maxfev)
    
    n = len(xdata)
    p = len(popt)
    
    # Calculate SSR
    y_pred = power1(xdata, *popt)
    SSR = np.sum((ydata - y_pred) ** 2)
    
    # Calculate SSR threshold. 95% CI
    SSR_threshold = SSR * (1 + scipy.stats.f.isf(alpha, 1, n - p) / (n - p) )
    
    # Preinitialize
    window_size = 0.99 # Scan exponents ranging from [1 - window_size, 1 + window_size]
    n_points = 1000
    exponents_left = np.linspace((1 - window_size)*popt[1], popt[1], n_points)
    exponents_right = np.linspace(popt[1], (1 + window_size)*popt[1], n_points)
    SSR_left = np.zeros((n_points, 1))
    SSR_right = np.zeros((n_points, 1))
    
    f_mod = lambda x, a: power1(x, a, e)
    
    # Go through left side of exponent
    for k, e in enumerate(exponents_left):
        
        popt_left, _ = curve_fit(f_mod, xdata, ydata, maxfev=maxfev)
        y_pred = f_mod(xdata, *popt_left)
        SSR_left[k] = np.sum((ydata - y_pred) ** 2)
        
    # Go through right side of exponent
    for k, e in enumerate(exponents_right):        
        popt_right, _ = curve_fit(f_mod, xdata, ydata, maxfev=maxfev)
        y_pred = f_mod(xdata, *popt_right)
        SSR_right[k] = np.sum((ydata - y_pred) ** 2)
        
    # Find the intersection points
    try:
        exp_lower_CI = exponents_left[ np.where(SSR_left  < SSR_threshold)[0][0]]
        exp_upper_CI = exponents_right[np.where(SSR_right > SSR_threshold)[0][0]]
        exp_CIs = [exp_lower_CI, exp_upper_CI]
    except IndexError: # CIs are very large; set as undefined
        exp_CIs = [np.nan, np.nan]
    
    # Plot if interested (useful for debugging/understanding)
    if plot:
        plt.plot(exponents_left, SSR_left)
        plt.plot(exponents_right, SSR_right)
        plt.axhline(SSR, color='k')
        plt.axhline(SSR_threshold, color='k')
    
    return exp_CIs

Define wrapper functions for curve_fit that returns both the 95% CIs and either $R^2$ or $R^2_{adj}$. See here for 95% CI reference (also cross-referenced with MATLAB and R's 95% CI calculations) and here for $R^2_{adj}$ reference

In [8]:
def adj_r_squared(r_squared, n, p):
    return 1 - (1 - r_squared) * (n - 1) / (n - p - 1)

def curve_fit_with_r_squared(f, xdata, ydata, maxfev=10000):
    popt, pcov = curve_fit(f, xdata, ydata, maxfev=maxfev)
    
    n = len(xdata)
    p = len(popt)
    
    if f.__name__ == 'power1':
        exp_CIs = calculate_power1_CIs_from_F_test(xdata, ydata)
    elif f.__name__ == 'power2':
        exp_CIs = calculate_power2_CIs_from_F_test(xdata, ydata)
    else:
        exp_CIs = None
        
    # Uncomment below line to use asymptotic, approximated CIs (not recommended)
    #CIs = tval(alpha, n, p) * np.diag(pcov)**0.5
    
    y_pred = f(xdata, *popt)
    r_sq = r2_score(ydata, y_pred)
    
    return popt, exp_CIs, r_sq

def curve_fit_with_adj_r_squared(f, xdata, ydata, maxfev=10000):
    popt, pcov = curve_fit(f, xdata, ydata, maxfev=maxfev)

    n = len(xdata)
    p = len(popt)
    
    if f.__name__ == 'power1':
        exp_CIs = calculate_power1_CIs_from_F_test(xdata, ydata)
    elif f.__name__ == 'power2':
        exp_CIs = calculate_power2_CIs_from_F_test(xdata, ydata)
    else:
        exp_CIs = None
        
    # Uncomment below line to use asymptotic, approximated CIs (not recommended)
    #CIs = tval(alpha, n, p) * np.diag(pcov)**0.5
    
    y_pred = f(xdata, *popt)
    r_sq = r2_score(ydata, y_pred)
    adj_r_sq = adj_r_squared(r_sq, n, p)
    
    return popt, exp_CIs, adj_r_sq

def plot_raw_data_subplot(data, labels, markers):
    
    x_max = 0
    
    # account for different types of raw data
    exponent = 1
    if labels.square_x:
        exponent = 2
    
    # legend tweaking based on number of series
    colon_str = ''
    if len(data) > 1:
        colon_str = ':\n'
    
    for k, dataset in enumerate(data):
        
        x = dataset[:,0]**exponent
        y = dataset[:,1]
        
        if labels.switch_xy:
            x, y = y, x
        
        x_max = np.max([np.max(x), x_max])

        popt1, CIs1, adj_r2_1 = curve_fit_with_adj_r_squared(t_onehalf, x, y)
        popt2, CIs2, adj_r2_2 = curve_fit_with_adj_r_squared(power1, x, y)
        popt3, CIs3, adj_r2_3 = curve_fit_with_adj_r_squared(power2, x, y)
                
        # Rounding
        prec2 = -int(np.floor(np.log10(popt2[1] - CIs2[0])))
        try:
            prec3 = -int(np.floor(np.log10(popt3[1] - CIs3[0])))
        except ValueError: # CI = nan
            prec3 = 0
        
        label = labels.legend_labels[k] + colon_str \
        + r'$' + labels.y_symbol_pred + '=%.3gt^{0.5} %+.3g$' % tuple(popt1) \
        + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_1 + '\n' \
        + r'$' + labels.y_symbol_pred + ('=%.3gt^{%.'+str(prec2)+'f }$') % tuple(popt2) \
        + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_2 + (', b=[%.'+str(prec2)+'f, %.'+str(prec2)+'f]') % tuple(CIs2) + '\n' \
        + r'$' + labels.y_symbol_pred + ('=%.3gt^{%.'+str(prec3)+'f } %+.3g$') % tuple(popt3) \
        + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_3 + (', b=[%.'+str(prec3)+'f, %.'+str(prec3)+'f]') % tuple(CIs3)
        
        # special case for carbon black data
        if popt3[0] > 1000:
            label = labels.legend_labels[k] + colon_str \
            + r'$' + labels.y_symbol_pred + '=%.3g t^{0.5} %+.3g$' % tuple(popt1) \
            + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_1 + '\n' \
            + r'$' + labels.y_symbol_pred + ('=%.3gt^{%.'+str(prec2)+'f}$') % tuple(popt2) \
            + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_2 + (', b=[%.'+str(prec2)+'f, %.'+str(prec2)+'f]') % tuple(CIs2) + '\n' \
            + r'$' + labels.y_symbol_pred + ('=%.4g t^{%.'+str(prec3)+'f} %+.4g$') % tuple(popt3) \
            + ', ' + r'$R_{adj}^2=%.4f}$' % adj_r2_3 + (', b=[%.'+str(prec3)+'f, %.'+str(prec3)+'f]') % tuple(CIs3)
        
        plt.plot(x, y, linestyle='None', marker=markers[k], label=label, color=colors[k])

    plt.xlabel(labels.x_label)
    plt.ylabel('{} {}'.format(labels.y_label, labels.y_unit))
    
    # Define standard xlim

def plot_residual_subplot(data, index, labels, xlim, marker):
    
    colors = cm.viridis(np.linspace(0, 1, 4))
    colors = colors[:,0:3]
    
    exponent = 1
    if labels.square_x:
        exponent = 2
    
    x = data[index][:,0]**exponent
    y = data[index][:,1]
    
    if labels.switch_xy:
        x, y = y, x

    popt1, CIs1, adj_r2_1 = curve_fit_with_adj_r_squared(t_onehalf, x, y)
    popt2, CIs2, adj_r2_2 = curve_fit_with_adj_r_squared(power1, x, y)
    popt3, CIs3, adj_r2_3 = curve_fit_with_adj_r_squared(power2, x, y)
    
    plt.axhline(0,color='tab:gray')
    
    plt.plot(x, y - t_onehalf(x, *popt1), marker, color = colors[2, :], label=r'$y=ax^{0.5}+b$')
    plt.plot(x, y - power1(x, *popt2), marker, color = colors[1,:], label=r'$y=ax^{b}$')
    plt.plot(x ,y - power2(x, *popt3), marker, color = colors[0,:], label=r'$y=ax^{b}+c$')
             
    y_label = 'Residual: '+ r'$' + labels.y_symbol +'-' + labels.y_symbol_pred + '$' + ' ' + labels.y_unit
    
    plt.xlabel(labels.x_label)
    plt.ylabel(y_label)
    plt.xlim(xlim)
    
    # annoying hack for Wright dataset
    legend_loc = 'best'
    if labels.dataset_name == 'Wright':
        legend_loc = 'lower right'
        
    plt.legend(title = labels.legend_labels[index], loc=legend_loc)

def plot_dataset(data, labels):
    
    # Find number of datasets
    n_datasets = len(data)
    
    # Initialize figure
    plt.figure(figsize=(fig_width*2, fig_height*2.5))

    # (a): Plot raw data
    ax = plt.subplot(221)
    plt.title('a', loc='left', weight='bold')
    xlim = plot_raw_data_subplot(data, labels, markers)

    # (b): Plot first series
    plt.subplot(223)
    plt.title('b', loc='left', weight='bold')
    plot_residual_subplot(data, 0, labels, xlim, markers[0])

    # (c): Plot last series
    plt.subplot(224)
    plt.title('c', loc='left', weight='bold')
    plot_residual_subplot(data, n_datasets-1, labels, xlim, markers[-1])

    plt.tight_layout()
    
    ax.legend(bbox_to_anchor=(0.55, 0.75), loc='center left', labelspacing=1.5,
              bbox_transform=plt.gcf().transFigure)
    
    # Save
    figure_filename = 'figures/{}.eps'.format(labels.dataset_name)
    plt.savefig(figure_filename, bbox_inches='tight', format='eps')

author = 'carbon black'
txts = glob.glob(author + '/*.{}'.format('txt'))
txts.sort(key=sortKeyFunc)
print(txts)

# Divide by masses to get mAh/g
mass = [0.855,0.930,0.952] # cells A, H, I respectively

# Switch order for more natural ordering
txts[0], txts[1] = txts[1], txts[0]
mass[0], mass[1] = mass[1], mass[0] 

time = []
SEI = []
CB_data_time = []
CB_data_cyclenum = []
CB_data_time_subfirstcycle = []
CB_data_cyclenum_subfirstcycle = []

for k, txt in enumerate(txts):
    d = np.genfromtxt(txt,skip_header=1)
    m = mass[k]/1000 # mg -> g
    
    #Q
    Q = d[:,2]/m # mAh -> mAh/g

    # Qd and Qc vs n
    peaks = find_peaks(Q)[0]
    Qpeaks = Q[peaks]
    Qd = Qpeaks[0::2]
    Qc = Qpeaks[1::2]
    
    # t (all cells cycle at C/20, with 1C = 200 mA/g -> C/20 = 10 mA/g)
    t_k = np.cumsum(Qpeaks)/10 # t = Q/I; (mAh/g / mA/g) -> h
    t_k = t_k[1::2]
    
    SEI_k = np.cumsum(Qd[:-1] - Qc)
    
    time.append(t_k) # time per cycle counts after discharge
    SEI.append(SEI_k)
    CB_data_time.append(np.vstack((t_k, SEI_k)).T)
    
    # Compute x as cycle number
    cyclenum_k = np.arange(len(SEI_k)) + 1
    CB_data_cyclenum.append(np.vstack((cyclenum_k, SEI_k)).T)
    
    SEI_k -= SEI_k[0]
    CB_data_time_subfirstcycle.append(np.vstack((t_k, SEI_k)).T)
    CB_data_cyclenum_subfirstcycle.append(np.vstack((cyclenum_k, SEI_k)).T)
    
    plt.plot(time[k],SEI[k],'o')
